# 知了贝商户端APP
知了贝为商户开发的app，功能包括门店管理，优惠券管理，使用quasar构建的安卓端app，IOS由于审核问题，目前使用quasar pwa构建
地图模块使用腾讯地图api，部分截图均取自于商户角色，商户下门店角色含有扫二维码核销功能，quasar的混合开发基于cordova，因此
调用相机和扫码功能均使用cordova插件调用原生功能
